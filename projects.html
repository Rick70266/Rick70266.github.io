專題製作展示

這裡記錄了我結合 SQL、Power BI 與 Python 的實戰案例。

零售業銷售數據分析系統

SQL Server Power BI DAX

專案描述： 針對連鎖店鋪的交易資料進行 ETL 清理，並建立星狀架構（Star Schema）資料模型。

  • 使用 SQL Server 進行資料彙整與視圖（View）建立。
  • 在 Power BI 中撰寫 DAX 度量值計算 YoY（年增率）與 MoM（月增率）。
  • 透過視覺化發現特定產品線在特定季節的毛利異常，協助減少 15% 的庫存堆積。

自動化房價爬蟲與預測分析

Python Pandas Selenium

專案描述： 開發 Python 腳本自動抓取房地產網站數據，並進行初步預測。

  • 使用 Selenium/BeautifulSoup 定時抓取最新成交行情。
  • 透過 Pandas 清洗異常值與缺失值。
  • 產出自動化 Excel 報表供決策參考。
