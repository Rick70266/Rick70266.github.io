關於我 (About Me)

您好！我是一名深耕於數據工程與商業智慧（BI）領域的專業人士。 我擅長建構穩健的資料底層架構，並透過數據建模與視覺化技術，將複雜的原始資料轉化為直觀、具備行動建議的決策儀表板。

我的核心價值在於連結「技術端」與「業務端」，不僅解決數據清理與 ETL 的工程難題，更能利用進階 DAX 運算挖掘數據背後的增長機會。

核心技術能力

SQL Server 架構

精通 T-SQL 編寫、預存程序優化及資料庫架構設計，確保數據讀取的高效與準確性。

Power BI 視覺化

擅長設計符合使用者習慣的商業報告，透過互動式介面提升跨部門數據溝通效率。

DAX 進階建模

運用 DAX 撰寫複雜的時間序列分析、動態度量值，精準定義 KPI 指標。

Python 自動化

利用 Python 進行資料抓取、前處理及統計分析，打造自動化數據流與模型部署。
